# 地缘经济政策图谱（公开网站）

公开访问地址：<https://feijiang425-maker.github.io/geoeconomic/>

这是无需登录的静态研究工作台。当前版本 `v2026.07.2` 包含168项政策事件、16个经济体和33条政策关系，提供时间线、国家比较、政策工具、政策强度和政策明细五种视图。

政策强度展示法律层级、覆盖范围、约束程度和实施范围的分项预编码，不是综合排名。标为“待第二编码”的记录尚未完成独立复核，不应直接用于最终实证结论。

## 政策强度打分规则

网页先对每项政策的四个维度分别赋1—5分，再对同一经济体具有有效分数的政策取简单平均。每条政策事件权重相同，缺失值不参与计算，也不以0分填补。

### 法律层级

- 5分：正式法律、议会立法；
- 4分：行政法规、行政命令、总统令或总统决定；
- 3分：部门规章、指令或正式国际协议；
- 2分：行政公告、联合声明或联合政策文件；
- 1分：战略、行动计划、指南、建议。

尚未正式通过的草案原则上保留缺失值，不按正式法律计分。

### 覆盖范围

- 5分：整体经济或跨多个主要部门；
- 4分：针对特定国家或经济体；
- 3分：覆盖一个完整行业或产业链；
- 2分：针对特定企业、项目、产品或技术；
- 1分：极小范围或试点事项，主要由人工判断。

当前自动映射为：整体性政策5分、国别政策4分、行业政策3分、企业／产品／技术政策2分。

### 约束程度

- 5分：具有禁止、许可、制裁、关税、列名或强制审查等高约束措施；
- 4分：其他具有法律约束力的政策；
- 3分：中等约束，需结合执行细则人工判断；
- 2分：约束性与倡议性并存的混合政策；
- 1分：非约束性战略、指南或自愿合作。

当前自动规则为：强制性限制／反制政策5分，其他强制性政策4分，混合政策2分，非约束性政策1分。

### 实施范围

- 5分：全球适用；
- 4分：区域实施；
- 3分：国内或双边实施；
- 2分：地方、局部或有限试点；
- 1分：尚无明确实施安排。

当前自动映射只使用全球5分、区域4分、国内或双边3分；1—2分需要人工判断。

### 为什么预算没有进入网页均值

预算存在币种、价格年份、期间、共享预算和宣布金额与实际支出差异。出口管制、制裁等高约束政策也通常没有可比预算。因此当前单独保存预算原币金额、期限和来源，不将其与其他维度简单加总。

### “需第二编码”是什么意思

它表示尚未完成独立第二编码的政策记录数。第二编码者需要独立复核四个核心维度，分歧由第三人裁决，并在完成一致率或Cohen’s κ检验后才能作为正式研究变量。

详细方法见项目文件 `docs/政策强度评分说明_v2.md`。

文件说明：

- `index.html`：页面结构；
- `style.css`：页面样式；
- `app.js`：筛选和视图逻辑；
- `policies.json`：公开政策事件数据；
- `relations.json`：公开政策关系数据。
