# 地缘经济政策图谱

“地缘经济政策图谱”是一个面向学术研究的主要经济体地缘经济政策检索、比较与证据追踪工作台。项目以政策事件为基本观察单位，系统整理2015年以来主要经济体在产业、技术、投资、贸易、能源与资源、供应链、数据与平台、国际合作、金融及基础设施等领域的地缘经济政策。

## 在线访问

网站为公开静态页面，访问者不需要登录 GitHub、ChatGPT 或其他账号。

## 主要功能

- 按关键词、经济体、政策领域、主要目的、有效状态和年份筛选政策；
- 通过时间线观察不同年份的政策发展；
- 比较主要经济体的政策数量与构成；
- 以矩阵形式展示政策领域与政策工具的组合；
- 查看政策名称、公布时间、生效时间、发布机构、政策摘要、政策目的、有效状态及官方来源；
- 标记尚待进一步核验的记录，便于持续开展证据追踪。

## 研究范围与口径

- **时间范围：** 2015年至今；
- **观察单位：** 具有独立政策含义的政策事件；
- **经济体范围：** 覆盖中国、美国、欧盟、日本及其他主要经济体；原则上以国家或单一经济体为单位。只有政策主要由区域联盟统一制定和实施时，才以区域联盟作为政策主体；
- **收录原则：** 根据实际政策活动及可获得的可靠证据收录，不刻意追求不同经济体政策数量一致；
- **政策分类：** 包括政策领域、政策工具、政策方向、战略目的、法律约束性和有效状态等维度；
- **证据来源：** 优先采用政府机关、立法机构、国际组织及其他权威机构发布的正式文件和网页。

## 数据与有效性说明

本项目属于持续更新的研究型数据库。页面显示的政策数量反映当前版本已经收集并编码的记录，不等同于相关经济体全部地缘经济政策的绝对数量。

“已核验”表示该记录的核心信息已依据现有来源进行核对；“状态待核验”表示政策状态、实施进展、终止时间或其他字段仍需进一步确认。核验标签反映证据整理进度，不代表对政策效果作出因果评价。

政策“有效性”主要指政策在法律或行政意义上的当前状态，例如已经生效、已经通过、仍处于提议阶段、已经暂停、修订或终止；现阶段不将其直接解释为政策是否达成经济、政治或安全目标。
